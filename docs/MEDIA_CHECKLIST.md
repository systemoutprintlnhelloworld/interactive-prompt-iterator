## 🎬 需要录制的 GIF 动图

### 1. 首次使用引导 (`docs/screenshots/demo-first-time-setup.gif`)
**场景**: 新用户首次使用应用的完整流程

**录制步骤**:
1. 打开应用首页（空白状态）
2. 点击右上角设置图标 ⚙️
3. 在"基础配置"标签页输入：
   - Base URL: `https://api.openai.com/v1`
   - API Key: `sk-...`（可以用测试预设）
   - Model: `gpt-4o`
4. 点击"测试连接"按钮
5. 显示"连接成功"提示
6. 点击"保存更改"
7. 关闭设置对话框
8. 在输入框输入"帮我写一篇技术文章"
9. 发送消息，展示 AI 响应

**时长**: 约 20-30 秒

---

### 2. 快捷搜索功能 (`docs/screenshots/demo-spotlight-search.gif`)
**场景**: 使用 Spotlight 搜索快速定位历史对话

**录制步骤**:
1. 在有多个历史对话的界面
2. 按下 `Ctrl+K` 快捷键
3. Spotlight 搜索框弹出（带动画效果）
4. 输入搜索关键词（如"React"）
5. 实时显示搜索结果（对话和收藏）
6. 点击某个搜索结果
7. 跳转到对应的对话

**时长**: 约 10-15 秒

---

### 3. 收藏功能演示 (`docs/screenshots/demo-favorites.gif`)
**场景**: 收藏提示词并从收藏列表快速应用

**录制步骤**:
1. 在某个对话中，AI 生成了提示词卡片
2. 点击卡片右上角的星标按钮 ⭐
3. 星标变为黄色，显示"已收藏"
4. 点击左侧边栏的"收藏"标签
5. 显示收藏列表
6. 点击某个收藏项
7. 展开显示完整内容
8. 点击"复制"按钮
9. 显示"已复制"提示

**时长**: 约 15-20 秒

---

### 4. 多语言切换 (`docs/screenshots/demo-i18n-switch.gif`)
**场景**: 切换界面语言和系统提示词

**录制步骤**:
1. 当前界面为中文
2. 点击右上角语言切换按钮（地球图标 🌐）
3. 选择"English"
4. 界面所有文字切换为英文（带流畅动画）
5. 打开设置对话框
6. 切换到"Prompt Management"标签
7. 显示系统提示词已自动切换为英文版本
8. 关闭设置
9. 再次点击语言切换，选择"简体中文"
10. 界面切换回中文

**时长**: 约 20-25 秒

## 📝 录制建议

### GIF 录制工具推荐:
- **Windows**: ScreenToGif（免费，功能强大）
- **macOS**: Kap（免费，简洁易用）
- **跨平台**: LICEcap（轻量级）

### 录制参数建议:
- **分辨率**: 1280x720 或 1920x1080
- **帧率**: 15-20 FPS（保持流畅且文件不过大）
- **文件大小**: 单个 GIF < 5MB（可以压缩）
- **鼠标指针**: 显示（帮助用户理解操作）

### 录制技巧:
1. 录制前清理浏览器缓存，确保界面干净
2. 使用测试数据，避免暴露真实 API Key
3. 操作速度适中，不要太快
4. 每个操作之间留 0.5-1 秒间隔
5. 录制完成后可以用 ezgif.com 压缩和优化
